# Policy & Compliance Management Backend System

A backend application designed to manage organizational policies, track policy versions,
and record employee acknowledgements for compliance purposes.

This system models real-world compliance workflows commonly used in enterprises,
with a strong focus on auditability and traceability.

---

## Features

- Create and manage organizational policies
- Maintain version history for each policy
- Record employee acknowledgements
- Track compliance activity through audit-friendly data models
- Clean, maintainable backend architecture

---

## Tech Stack

- Python
- FastAPI
- SQLAlchemy
- SQLite (or PostgreSQL)
- REST APIs
